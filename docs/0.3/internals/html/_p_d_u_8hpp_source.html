34  /**
35  * \internal
36  *
37  * \brief The base class of all PDU's
38  *
39  * This class is never instantiated itself, but serves as the base class
40  * for concrete %PDU classes which don't have a context field (e.g.
41  * OpenPDU). For concrete %PDU's with context field, the class
42  * PDUwithContext is used.
43  *
44  * This class represents the %PDU header and contains data and
45  * functionality common to all %PDU types. It provides an automatic
46  * packetID generator. Whenever a PDU object is created, the packetID is
47  * automatically incremented. The two exceptions are:
48  * - The parse contructor fills the packetID from the received %PDU
49  * - The ResponsePDU class overwrites the packetID
50  *
51  * The mechanism uses a static class member packetID_cnt to store the last
52  * used packetID. The counter wraps at its limit.
53  */

variable *  v 
)

Create a VarBind with an oid and a var.

The variable must be one of the following types:

  • Integer
  • Octet_String
  • oid
  • IpAddress
  • Counter32
  • Gauge32
  • TimeTicks
  • Opaque
  • Counter64 If the type of the variable cannot be determined, inval_param is thrown.

Create VarBind without var, but with a type.
